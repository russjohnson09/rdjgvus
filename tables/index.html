Contact table demo

Generate contacts using the generate contact button or create your own by filling out the contact form. Tablesorter.js is used to sort tables. Contacts can be deleted by selecting them and using the Del key. Right now adding contacts does not update the table and a refresh is required. Contacts are stored using IndexedDB meaning they are stored locally by your browser. I don't think this works on IE and if you are using it consider upgrading. Sort on multiple columns by holding shift and selecting multiple columns.
